# Tic Tac Toe Championship

A modern Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript** featuring player names, score tracking, and a championship mode where the first player to reach 5 wins becomes the champion.

## Features

* Two-player gameplay
* Custom player names
* Live scoreboard
* Winner detection
* Draw detection
* Next Round functionality
* New Match option
* First-to-5-wins Championship Mode
* Responsive and user-friendly interface

## How to Play

1. Enter names for Player X and Player O.
2. Click **Start Game**.
3. Players take turns placing X and O on the board.
4. Win a round by completing a row, column, or diagonal.
5. The winner of each round earns 1 point.
6. The first player to reach **5 points** becomes the Champion.
7. Use **Next Round** to continue the tournament.
8. Use **New Match** to reset scores and start over.
